WingmanServer Usage Documentation


WingmanServer - Server program for Logitech Wingman 3D joysticks

This is a server for the following Logitech (Vendor ID: 0x046D) joysticks
1. Wingman Extreme 3D Digital (Product ID: 0xC212)
2. Force3D (Product ID: 0xC283) (no force feedback support)
3. Extreme 3D Pro (Product ID: 0xc215)

usage:
To use the joystick please ensure that the USB manager (devu-uhci or
devu-ohci) and manager for HID devices (io-hid) are running. Here are
the commands (you may put these in /etc/rc.d/rc.local)

/sbin/devu-uhci & (or /sbin/devu-ohci &, whatever works)
/sbin/io-hid &
mount -Tio-hid devh-usb.so &

A server named "usbjs" can then be started as follows :

WingmanServer -n qrts/usbjs &

After starting the server, activate any of the degrees of freedom of
the joystick once so that data structures are initialized to proper
values.
